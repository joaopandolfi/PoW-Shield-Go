2. **WAF consumes body without restoring** (`server/web/middleware/waf.go:93`) ✅
   - Already fixed: body restored with `r.Body = io.NopCloser(bytes.NewReader(body))`

### P1 - Should Fix (Major)

3. **In-memory cache buffer overflow** (`server/internal/cache/memory.go:44`)
   - 150-entry limit crashes service under moderate load
   - Fix: Implement TTL-based eviction or increase limit significantly

4. **Hardcoded CookieStore secret** (`server/config/config.go:123`)
   - Security risk in production
   - Fix: Generate random key or enforce environment variable

5. **TLS verification disabled** (`server/internal/request/request.go:20`)
   - Man-in-the-middle vulnerable proxy
   - Fix: Remove `InsecureSkipVerify: true` or configure CA cert
